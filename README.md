Инструкция по использованию

Веб-справочник доступен в сети Интернет по адресу: http://brigada-kanbanov.ru.

### Для пользователей справочника:
1. Просмотр терминов. На главной странице отображается алфавитный список всех терминов. Используйте пагинацию внизу страницы для перехода между страницами.
2. Поиск. В верхней части страницы находится строка поиска. Введите ключевое слово (например, «винт» или «админ») и нажмите «Найти». Поиск работает по названию и определению, регистр букв не имеет значения.
3. Переход к ГОСТ. Рядом с определением термина находится ссылка «Название в соответствии с ГОСТ». Нажав на неё, вы перейдёте на сторонний ресурс с официальным определением термина.
4. Сброс поиска. Чтобы вернуться к полному списку, нажмите кнопку «Сброс» рядом со строкой поиска.

Для администратора:
1. Вход в панель управления. Перейдите по адресу `brigada-kanbanov.ru/admin/` и введите логин и пароль суперпользователя.
2. Управление терминами. В разделе «Термины» вы можете добавлять новые записи, редактировать существующие определения и ссылки на ГОСТ, а также удалять устаревшие термины.
3. Массовый импорт. Для быстрой загрузки большого количества терминов из Word-документа используется консольная команда `python manage.py import_terms_docx terms.docx`.
